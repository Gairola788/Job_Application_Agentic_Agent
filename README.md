- Resume Parsing
- Job Description Analysis
- Skill Matching
- Resume Improvement
- Cover Letter Generation
- Interview Question Generation
- RAG-based Resume Chat
- Streamlit UI

- LangGraph
- LangChain
- Groq
- Streamlit
- PyPDF
- FastAPI (if used)
